🚗 ESP32 Smart Parking System
📌 Deskripsi

Project ini merupakan sistem Smart Parking berbasis ESP32 yang memanfaatkan sensor ultrasonik untuk mendeteksi keberadaan kendaraan. Sistem ini mampu melakukan monitoring dan controlling secara real-time menggunakan protokol MQTT serta penyimpanan data melalui HTTP API (MockAPI).

Data yang dihasilkan seperti jarak dan status parkir akan ditampilkan pada website dalam bentuk dashboard interaktif, lengkap dengan indikator warna dan histori monitoring.
